# MCADV Website

This is the static website for the MCADV project, styled as a medieval bulletin board.

## Files

- `index.html` - Main webpage with project philosophy and ethos
- `style.css` - Medieval-themed dark styling

## Deployment

This website is designed to be deployed on Netlify. It will automatically deploy on git pull.

## Design Philosophy

The website uses:
- Dark medieval/bulletin board aesthetic
- Gothic and medieval fonts (MedievalSharp, Cinzel, UnifrakturMaguntia)
- Parchment-like textures and borders
- Non-technical, philosophy-focused content
- Emphasis on decentralization, community, and open source values

## Local Testing

To view locally, simply open `index.html` in a web browser.
