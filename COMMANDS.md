# Command Handling

## Supported Command

The only command supported for starting a new collaborative adventure is:

`!start [theme]`

## Available Themes
- fantasy
- medieval
- scifi
- horror
- dark_fantasy
- urban_fantasy
- steampunk
- dieselpunk
- cyberpunk
- post_apocalypse
- dystopian
- space_opera
- cosmic_horror
- occult
- ancient
- renaissance
- victorian
- wild_west
- comedy
- noir
- mystery
- romance
- slice_of_life
- grimdark
- wholesome
- high_school
- college
- corporate
- pirate
- expedition
- anime
- superhero
- fairy_tale
- mythology

## Note
Support for the `!adv` command has been completely removed, and all user-facing help texts and documentation have been updated accordingly.
